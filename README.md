# MGPIC 2026

Official website for MoonBit MGPIC 2026.

## Tech stack

- MoonBit
- Rabbita (`moonbit-community/rabbita@0.11.8`)
- Vite
- GitHub Pages

## Features in this refactor

- Chinese / English bilingual content
- Light / Dark theme toggle
- Information architecture aligned with an open source program guide
- GitHub Pages deployment workflow

## Development

Install dependencies:

```bash
npm i
```

Run locally:

```bash
npm run dev
```

Build:

```bash
npm run build
```

## MoonBit dependency

```bash
moon add moonbit-community/rabbita@0.11.8
```

## Deployment

The repository includes a GitHub Actions workflow for GitHub Pages.
